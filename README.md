# Altium — Web

Web corporativa de **Altium**: trabajos verticales y en altura, rehabilitación de fachadas, impermeabilización con poliuretano, bajantes de PVC y reformas en Barcelona y alrededores.

Sitio estático (HTML + CSS + JS, sin dependencias). Publicado con **GitHub Pages**.

## Páginas
- `index.html` — Inicio
- `trabajos-verticales.html` — Trabajos verticales y en altura
- `reformas.html` — Reformas e impermeabilización
- `nosotros.html` — Sobre Altium
- `contacto.html` — Contacto (formulario con Web3Forms)
- `politica-cookies.html` — Política de cookies y privacidad

## Estructura
- `assets/styles.css` · `assets/app.js` — estilos y JS compartidos
- `assets/img/` — fotos del sitio · `assets/logo.png` — logotipo
- `sitemap.xml` · `robots.txt` — SEO
